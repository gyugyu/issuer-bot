# Issuer-bot

this is GitHub Issuer Create bot for Slack.

## Endpoint
- /register
- /deregister
- /list


## Using Slack API
- chat.postMessage
- conversations.info
- conversations.history
- conversations.replies
- users.info
- users.list
- groups.list

## Install
1. Create GitHub Personal Access Token.
2. Create Slack Application.
3. Set Values to manifests
4. Apply manifests
```
$ kubectl -f manifests
```
